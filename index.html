TWORZE NOWOCZESNE STRONY INTERNETOWE

Projektowane z myślą o tobie

Każda strona tworzona jest głównie z myślą dostarczenia użytkownikowi jak najbardziej funkcjonalnej strony dla niego. Staram się wejść w role użytkownika i zrozumieć jego potrzeby ,wymagania lub te aspekty których oczekuje od strony internetowej. Dlatego kontakt z klientem, zrozumienie oraz analiza jego potrzeb to podstawa budowy dobrej witryny internetowej która sprosta wymaganiom klienta.

PODĄŻAJĄCE ZA NAJNOWSZYMI TRENDAMI W DESIGN

Czasy kiedy strony miały tylko "działać" odeszły w odmęty przeszłości. Dzisiejsze strony internetowe często zaczynają przybierać formę dzieł sztuki. Warstwa wizualna strony ma ogromny wpływa na odbiór całościowy witryny przez użytkownika .W swoich projektach staram się czerpać to co najlepsze z nowoczesnego web designu w miarę możliwości próbować tworzyć własne rozwiązania.

Wykorzystujące popularne języki programowania

Witryny internetowe które tworze korzystają z najnowszy rozwiązań dostarczonych przez obecne dzisiaj języki programowania. W swoich projektach korzystam z takich rozwiązań jak HTML 5 ,CSS3 czy JavaScript ES5 z ES6 włącznie. Wykorzystuje także języki Back-Endowe jak PHP czy Python a w przyszłości także Node.js. Przy projektach korzystam z preprocesorów SCSS/SASS oraz biblioteki JQuery a w przyszłości także z React.js

Technologie które obecnie poznaje

HTML 5

Poznałem podstawy tworzenia strona za pomocą HTML oraz zasady tworzenia semantycznego kodu. Rozumiem zasadę działania modelu pudełkowego i mechanizmy manipulowania elementami za pomocą takich właściwości jak display czy position oraz posiadłem umiejętność tworzenia takich elementów strony jak slidery ,menu typu hamburger czy okna popup.

Obecny poziom znajomości technologii:

CSS 3

Opanowałem zagadnienia manipulowania wyglądem elementów od podstawowych typu color czy font-sizie do bardziej zaawansowanych jak position, display czy box-shadow. Umiejętnie posługuje się kaskadowością stylów oraz rozumiem działanie pseudo klasy i pseudo elementów. Poznałem podstawy pracy z flex-box oraz css-grid. Nauczyłem się także tworzenia stron responsywnych wykorzystujący mediaqueries oraz flex-box oraz prostych animacji korzystających transition ora keyframes.

Obecny poziom znajomości technologii:

JavaScript

Poznałem zagadnienia oraz zasady programowania strukturalnego w JavaScript włączając w to podstawy języka od zmiennych przez pętle po tablice i obiekty. Manipulacje obiektem DOM za pomocą JavaScript, wywoływanie funkcji za pomocą eventów, walidacji danych wejściowych oraz modyfikacji danych wyjściowych oraz postawy poprogramowania zorientowanego obiektowo.

Obecny poziom znajomości technologii:

PHP

Nauczyłem się składowych elementów języka także tworzenie sesji czy cookies. Budowania połączenia z bazą danych oraz zarzadzanie nimi. Wykorzystywanie połącznia z baza do pobierania aktualizowania, wstawania czy usuwania rekordów z bazy danych. Do tych operacji wykorzystuje kwerendy tworzone za pomocą języka SQL.

Obecny poziom znajomości technologii:

Git i GitHub

Rozumiałem mechanizmy rządzące systemem kontroli wersji Git oraz działanie serwisu GitHub. Poznałem podstawy pracy z systemem od tworzenia repozytoriów ,dodawania plików do stage area zarządzania commitami i ich wysłania przy pomocy metody push oraz pobierania z pomocą pull. Potrafię także połączyć githuba z edytorem IDE oraz wykorzystać wiersz poleceń BASH do zarządzania systemem.

Obecny poziom znajomości technologii:

Sass oraz Scss

Tworze style przy pomocy SASS oraz rozpoznaje różnice miedzy SASS a SCSS. Umiejętnie posługuje się mechanizmami typu mixin, zmienne oraz import.

Obecny poziom znajomości technologii:

JQuery

Potrafię manipulować elementami strony za pomocą składni JQuery ,tworzyć i usuwać elementy strony wykorzystując do tego eventy oraz przy pomocy tej biblioteki walidować formularze.

Obecny poziom znajomości technologii:

Python

Umiem wykorzystywać postawy języka jak pętle, zmienne czy funkcje oraz manipulować danymi za pomocą składni python. Potrafię wyodrębniać dane z plików i wykorzystywać je do obliczeń działań itp oraz tworzyć nowe lub modyfikować istniejące już pliki w systemie.

Obecny poziom znajomości technologii:

Technologie które mam zamiar poznać w niedalekiej lub dalszej przyszłości

React.js

Jedna z najbardziej znanych i lubianych bibliotek javscript. Jej poznanie pozwoli mi lepiej budować strukturę kodu oraz utrzymywać w nim większą przejrzystość. Umożliwi mi w szybszy sposób budowania interfejsów użytkownika a z czasem nawet aplikacji mobilnych.

Node.js

Obecnie od większość Front-End Deweloperów wymaga się znajomości chociaż jednego języka back-endowego. .Poznanie tego narzędzia umożliwi mi większą kontrole nad projektem i pozwoli mi lepiej zrozumieć machizmy działające pod warstwą wizualną strony.

Less

Jedne z preprocesorów CSS. Umiejętność posługiwania się SASS oraz LESS pozwoli mi bez większych problemów pracować w różnych zespołach ,przy różnych projektach w których np. wykorzystuje się tylko LESS a w innych tylko SASS lub SCSS. Przyspieszy też moja prace z kodem CSS .

Adobe Photoshop

Photoshop jest często wykorzystywany prze web deweloperów do projektowania leaytów strony, poprawiania zdjęci czy tworzenia lub poprawiania grafik na potrzeby strony. Opanowanie tego narzędzia przyspieszy moja prace z projektowanie witryn oraz pozwoli bardziej rozwinąć umiejętności zwązane z web designem.

WordPress

WordPress CMS to jeden z najpopularniejszych systemów zarządzani treścią na świecie. Poznanie mechanizmów oraz metod pracy z tym narzędziem pozwoli tworzyć, administrować lub modyfikować strony oparte o ten CMS.

Bash i system Linux

Choć obecnie pracuje na systemie Windows potrafię dostrzec przewagi sytemu Linux na systemem Windows w pewnych aspektach. System ten jest bardzo przyjazny dla wszelkiego rodzaju twórców treści. Istnieją specjalne dystrybucje tego systemu skierowane wprost do programistów a wiersz poleceni systemu oparty na BASH znacząco przyspiesza prace z narzędziami developerskimi.
